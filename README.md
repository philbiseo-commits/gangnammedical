# Gangnam Medical

Static English landing page for a Korean plastic surgery clinic discovery MVP.

## Included

- `index.html` - homepage
- `styles.css` - visual system and layout
- `script.js` - renders the Top 10 clinic cards

## Positioning

- Top 10 format without explicit 1-10 ranking
- Card-based discovery inspired by travel/activity platforms
- Built for international patients
- Includes trust section, FAQ, and clinic representative contact prompt
